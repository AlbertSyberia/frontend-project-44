# Игра "Проверка на четность"

Простая консольная игра, в которой необходимо ввести "yes" или "no" на вопрос является ли данное число четным или же нет. Если ввести любое другое значение программа будет считать что его ошибочным.

# Игра "Калькулятор"

Простая консольная игра, в которой пользователю показывается случайное математическое выражение, например 35 + 16, которое нужно вычислить и записать правильный ответ.

# Игра "НОД"

Простая консольная игра, в которой пользователю показывается два случайных числа, например, 25 50. Пользователь должен вычислить и ввести наибольший общий делитель этих чисел.
